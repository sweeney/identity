SPA OAuth Demo

Single-page app using OAuth 2.0 Authorization Code + PKCE. No backend — pure JavaScript + localStorage.

How it works

  1. Click "Sign in" — generates a PKCE code verifier and redirects to the Identity server
  2. Log in on the Identity server (this app never sees your password)
  3. Identity server redirects back with an authorization code in the URL
  4. This page exchanges the code + verifier for tokens via JavaScript fetch()
  5. Tokens are stored in localStorage

Session Ended

Your session was ended — the token could not be refreshed. This happens when your account is deactivated, the refresh token expires, or token theft is detected.
